Some instructions on cleaning the dataset:

  1. Delete those questions that are too subjective to give a specific answer.
  2. Edit alternatives that are not related to the question.
    • Alternatives should be related to questions.
    • Alternatives shouldn't be similar to the correct answer, i.e. not to cause confusion.
    • Examplar after cleaning:
    • Q): Why does the mixture look white? A): Milk is in it.
    • And the alternatives are: A) Because the light is strong. B) Everything in it is white. C) Because it is covered with white cloth. D)
  3. Each question must have one correct answer and FOUR alternatives. For 'yes/no' questions, we do not need to edit them as long as there is both 'yes' and 'no'.
  4. There are some questions where you may not come up with enough reasonable alternatives, then you just fill the rest with some words under the same category, or something else you can think of.
  5. A special rule for 'when', 'at what time' questions:
    • The answer often comes with "at time 423", "456 seconds", "123 - 147", etc. We want them to have numbers only. So you need to edit them to "423", "456", "123"(just keep the beginning of the time period)
    • This is the ONLY situation where you should edit the correct answer.
    • The number here stands for the seconds in a video. The longest could be more than 10 min, which we take it as up to 700 seconds. Please make up alternatives randomly within the reasonable range.

Submission:

  1. If the question is too subjective, check the checkbox before "Delete"
  2. If you need to modify the correct answer to a certain form, check the checkbox before "Edit answer" and type the correct answer in the corresponding textbox on the same row.
  3. If you need to modify the alternatives, check the checkbox before "Edit alternatives" and type the new alternatives in the corresponding textbox on the same row.
  4. If the QA pair is good, do nothing, and then submit.
  5. Click !
